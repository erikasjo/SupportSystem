# SupportSystem

Ett supportsystem där elever, lärare och administratörer kan rapportera tekniska problem.

## Funktioner
- Skapa nya supportärenden
- Visa alla tickets
- Filtrera tickets efter status
- Uppdatera status
- Ta bort tickets

## Tekniker
- React (frontend)
- ASP.NET Core Web API (backend)
- Fetch API för kommunikation

## Hur man kör projektet

### Backend (API)
1. Starta projektet i Rider/Visual Studio
2. API körs på:http://localhost:5090/api/supporttickets


### Frontend (React)
1. Öppna terminal i projektet
2. Kör:
   npm install
   npm rund dev
3. Öppna:http://localhost:5173/
